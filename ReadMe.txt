Download Node.js Here -----> https://nodejs.org/en/download/

-----How to run the server-----
1. Open terminal, command prompt, powershell 

2. Run the Command
	node ./server.js IP PORT api-key homeDir

-----------------------------------

WHERE
IP - is the IP you want the server to run on use 127.0.0.1 
PORT - 8081
api-key - api key of the watson visual recognition
homeDir - is the home directory where the server files are stored EX: C:\Users\Asher\Desktop\Work\JS\web\WatsonTestServer\

------TO USE-------

Once the server is started go to 127.0.0.1:8081/demo in a web browser
